# 🚀 User Management API

Esta es una API RESTful construida con **Node.js** y **Express**, diseñada para gestionar usuarios y conectarse a una base de datos **PostgreSQL**. El proyecto está estructurado para ser desplegado fácilmente y versionado en GitHub.

## 📋 Características

* **Registro de usuarios:** Almacenamiento seguro en base de datos.
* **Validación de datos:** Control de correos únicos y campos obligatorios.
* **Arquitectura Escalable:** Uso de Express para un manejo de rutas eficiente.
* **Base de Datos Relacional:** Estructura SQL optimizada.

---

## 🛠️ Tecnologías Utilizadas

* **Node.js** - Entorno de ejecución.
* **Express.js** - Framework para la API.
* **PostgreSQL** - Sistema de gestión de base de datos.
* **Git/GitHub** - Control de versiones.
